# nno
database backed link shortener

## Usage
```
$ nno -h
Usage of nno:
  -db string
        database file (default "./nno.db")
  -host string
        host to listen on (default "localhost")
  -port int
        port to listen on (default 8080)
```

## Installation
```
$ go install github.com/delta9000/nno
```
